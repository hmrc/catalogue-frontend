Creating prototype repository: @repoName

The prototype repository @repoName is being added to the catalogue. This can take around 30 minutes to appear and for all catalogue features to be available.

What to do next?

  • Check the prototype repository has been created in GitHub (this can take around 5 minutes)
  • Check the prototype has been deployed to Heroku
  • Check the status of the prototype is 'Running'
  • Prototypes are turned off in Heroku throughout the weekend in order to save resources, if you need to access the prototype you will need to restart it:
    • By navigating to your prototype in Heroku and clicking on the 'Restart prototype' button
    • Alternatively, you can restart from the catalogue by navigating to the repository page and clicking the 'Start' button

Helpful links

  • Prototype user guide for additional information around prototyping and Heroku
  • Prototype community slack for general discussions and troubleshooting prototypes
