About the catalogue

The catalogue is a resource you can use to find information about teams working on the Tax Platform, and the services and libraries they own.

You can browse the lists (which are searchable using your browser) and drill down to access links to Jenkins, Travis & Github for each service or library belonging to that team)

This service is currently in an alpha phase, and needs your feedback in order to improve. If you have anything to offer, or find something wrong, you can contact us in the #team-platops room in Slack.

How is the catalogue maintained?

We use team and repository information from Github open and Github Enterprise in order to programatically build the catalogue. If you notice something is not quite right, you can usually fix it by updating Github directly.

Currently we only show data for repositories that contain services. We consider a repository as a service if we find either an app folder, or a Procfile in its root directory.

Some teams and repositories are hidden, for example microservice templates that have the structure of play apps but are not services themselves. This list is maintained in app-config-base. If you want a repository or team hidden, you can send us a pull request.

We reload the catalogue once every hour, so you may need to wait a little while until your changes take effect.
